Agora

This website was made as a final project for my Web Authoring II class at Broward College. Agora is a prototype chatroom site which features both a public site section to learn more about the site, as well as a private site section which features a variety of chatrooms - accessed via a JavaScript login feature.

As a prototype, this site's chatrooms are no longer live, and all user login information was manually coded into the site via JavaScript. However, you can check out the public section of this site here.

This website was made from scratch in Visual Studio Code, without the use of any prebuilt templates. The background animation for this site's landing page was made by me in Adobe After Effects, and a number of this site's background images were edited by me in Adobe Photoshop.

To check out a closeup the animation I created for this site, be sure to visit the After Effects section of the gallery. To view all of the Photoshop content I created for this site, be sure to visit the Photoshop section of the gallery.
